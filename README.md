# Decentralized Platform for Interactive Music Performance with Parallel Distributed and High Performance Computing

This repository contains an academic project exploring the use of artificial intelligence (AI) in contemporary popular music production.

The work examines how AI technologies are designed, adopted, and applied in real-world music production workflows, with a focus on creative practice, studio processes, and emerging tools.

The submission folders are made structure progress research, as per the course instructors guidance.
